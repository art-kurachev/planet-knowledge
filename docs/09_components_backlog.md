# Components Backlog

Реестр компонентов, которых не хватает в дизайн-системе.

**Статусы:** `нет` — не начат / `в работе` — проектируется / `готово` — добавлен в DS

---

## 🔴 Критично

| Компонент | Приоритет | Статус | Описание |
|-----------|-----------|--------|----------|
| Modal / Dialog | 🔴 Критично | нет | Подтверждения, согласования, destructive-действия. Без единого стандарта каждый попап собирается руками из PopUpComponents |
| Empty State | 🔴 Критично | нет | Пустой реестр, пустой результат поиска, пустой раздел. Базовая UI-ситуация, сейчас решается произвольно везде |

## 🟠 Важно

| Компонент | Приоритет | Статус | Описание |
|-----------|-----------|--------|----------|
| Sidebar / Drawer | 🟠 Важно | нет | Правая/левая панель для карточки сущности (задача, счёт, договор). Один из самых частых паттернов в CRM |
| Pagination | 🟠 Важно | нет | Постраничная навигация для реестров и таблиц. Таблицы есть — пагинации нет |

| Breadcrumbs | 🟡 Нужно | нет | Навигационный хлебный путь для глубоких разделов (закупки → запрос → счёт → позиция) |
