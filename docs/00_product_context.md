# Product Context

## Продукт

**Planet CRM** — внутренняя веб-CRM/ERP система для строительной компании (жилые комплексы, Москва).  
Рабочая операционная среда, используется ежедневно 8+ часов.

## Модули

- Закупки и снабжение — запросы, счета, сравнение поставщиков
- Финансовые процессы — согласование, оплата
- Договорная работа
- Управление задачами — канбан, статусы, дедлайны
- Отчётность — для руководства
- CMS — управление сайтами
- CRM — работа с клиентами, недвижимость

## Платформа

- Web-only, светлая тема (Dark mode — в разработке)
- Синяя визуальная айдентика
- Интерфейс насыщенный, но структурный

## Среда использования

- 6–15 вкладок одновременно
- Параллельно мессенджеры и звонки
- Работа кусками по 5–20 минут с частыми прерываниями
- Возвраты в интерфейс после отвлечений

**Следствия для UX:**
- Интерфейс устойчив к прерыванию
- Состояния очевидны без «вспоминания»
- Нельзя требовать удержания сложного контекста в голове

## Цена ошибки

Ошибка = деньги, срыв сроков, конфликт с руководством. Поэтому:

- Критические действия не маскировать
- Кнопки однозначны
- Destructive-действия не ставить рядом с подтверждающими
- Подтверждение — осмысленное, не декоративное

## Принцип нагрузки

Информационная плотность ≠ визуальный шум.  
Система может быть насыщенной данными, но структура всегда очевидна.

## Приоритеты развития

1. Снижение времени принятия решения
2. Повышение читаемости статусов
3. Уменьшение когнитивной нагрузки
4. Предсказуемость навигации
5. Контроль и прозрачность
