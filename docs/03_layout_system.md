# Layout System

> Отступы кратны 8. Единственный шрифт — Inter (400, 500, 600).
> Токены цветов — `docs/COLOR_TOKENS.md`.

---

## Сетка отступов

Допустимые значения: `4, 8, 12, 16, 24, 32, 40, 48, 56, 64`

- Основные отступы между блоками — кратны 8
- Внутренние отступы компонентов — см. `docs/04_components.md` (Size Tiers)
- При выборе между `12` и `16` — выбирать `16`
- `4` — только для плотных компонентов (Little tier)

---

## Брейкпоинты

| Breakpoint | Ширина | Поведение |
|------------|--------|----------|
| Desktop | ≥ 1280px | Основной режим. Все колонки видны |
| Tablet | 768–1279px | Боковая панель сворачивается или скрывается |
| Mobile | < 768px | Минимальная поддержка (не приоритет) |

> Минимальная поддерживаемая ширина: **768px**. Медиа-запросы — только в CSS-файле модуля, не в inline-стилях.

---

## Структура страницы

```
[Sidebar навигация] | [Основной контент]
     240px (fixed)  |  flex: 1, overflow-y: auto
```

- Фон страницы: токен `bg/page` (`#EFF1F8` Light / `#1E212B` Dark)
- Фон карточек/панелей: токен `bg/surface` (`#FFFFFF` Light / `#2D3243` Dark)
- Sidebar: фиксированная ширина, не участвует в скролле контента
- Контентная область: `minHeight: 100vh`, скролл только внутри неё
