# Constraints

## Технические ограничения

### Шрифт
- **Inter** — единственный допустимый шрифт
- Доступные начертания: Regular, Medium, Semi Bold

### Figma API

1. **Всегда использовать стили** — `await node.setFillStyleIdAsync(styleId)`, никогда не хардкодить hex-значения
2. **Async API** — использовать `figma.getNodeByIdAsync()`, не `figma.getNodeById()`
3. **После `arrange_component_set`** — ID вариантов меняются; всегда заново читать `set.children` перед применением стилей
4. **Иконки** — стиль ставить только на Vector внутри Instance, не на сам Instance-фрейм
5. **Стили иконок** — в табах и похожих компонентах иконки используют `Secondary/BlueGray` в обоих состояниях (active и inactive)
