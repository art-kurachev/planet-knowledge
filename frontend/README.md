# Planet Frontend — Тест компонентов

Папка для сборки React-компонентов из Figma и их визуальной проверки.

## Запуск

```bash
cd frontend
npm install
npm run dev
```

Открой http://localhost:5173

## Структура

```
src/
  components/       # React-компоненты
    primitives/     # Badge, Tag, HR, VR, BtnPDF, IconDanger, IconArrow
    table/          # TH, TD
    BarChart.jsx
    ProjectsPanel.jsx
    EmployeesPanel.jsx
    TabelPanel.jsx
    SKUDBody.jsx    # Сводный компонент (СКУД)
  tokens/           # Цвета из Figma
  data/             # Мок-данные
```
